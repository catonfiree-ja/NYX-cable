สายคอนโทรล

สายไฟคอนโทรลคุณภาพยุโรป สำหรับงานระบบอัตโนมัติ ตู้ควบคุม เครื่องจักรในโรงงานอุตสาหกรรม

10 รุ่นในหมวดนี้

YSLY-JZ : สายคอนโทรล

YSLY-JZ

สายคอนโทรลมาตรฐานยุโรป DIN VDE ตัวนำทองแดงฝอยละเอียด ฉนวน PVC อ่อนตัวสูง 300/500V

Olflex Classic 110 : สายคอนโทรล

OLFLEX-110

สายคอนโทรลพรีเมี่ยมจาก LAPP ตัวนำทองแดงฝอยละเอียด ฉนวน PVC 300/500V มาตรฐาน VDE

JZ 500 Volt : สายคอนโทรล

JZ-500

สายคอนโทรล 500V ตัวนำทองแดงฝอย ฉนวน PVC คุณภาพสูง เหมาะสำหรับงานอุตสาหกรรมทั่วไป

OPVC-JZ : สายคอนโทรล

OPVC-JZ

สายคอนโทรล OPVC ราคาคุ้มค่า ตัวนำทองแดงฝอย ฉนวน PVC 300/500V

Flex-JZ : สายคอนโทรล

FLEX-JZ

สายคอนโทรลอ่อนตัวนุ่ม ตัวนำทองแดงฝอยละเอียดพิเศษ เดินสายง่ายในพื้นที่แคบ

CVV / CVV-F / CVV-S : สายคอนโทรล

CVV

สายคอนโทรลมาตรฐาน JIS ใช้งานกันอย่างแพร่หลายในประเทศไทย 600V ทนความร้อน 70°C

VCT : สายไฟอ่อน

VCT

สายไฟอ่อน VCT มาตรฐาน มอก. ตัวนำทองแดงฝอย ฉนวน PVC 450/750V เหมาะงานทั่วไป

Multicore Cable : สายมัลติคอร์

MULTI

สายไฟมัลติคอร์หลายแกน เหมาะสำหรับงานคอนโทรลและจ่ายไฟเครื่องจักรอุตสาหกรรม

YSLY-JZ 1kV : สายคอนโทรล 1000V

YSLY-1KV

สายคอนโทรลรับแรงดัน 1000V สำหรับงาน Power ที่ต้องการความอ่อนตัวสูง ใช้แทนสาย NYY ได้

Control Cable : สายคอนโทรล (ภาพรวม)

CTRL

รวมข้อมูลสายคอนโทรลทุกรุ่น วิธีเลือก คุณสมบัติเปรียบเทียบ สำหรับงานอุตสาหกรรม

สายคอนโทรลคืออะไร? รู้รอบด้านพร้อมวิธีเลือก

สายคอนโทรลเป็น "หัวใจ" ของระบบ Automation ในโรงงานอุตสาหกรรม ใช้สำหรับส่งสัญญาณควบคุม (Control Signal) ระหว่างอุปกรณ์ต่างๆ เช่น PLC, Sensor, Motor Drive เน้นความแม่นยำของสัญญาณ ไม่ใช่พลังงานไฟฟ้าแรงสูง

เปรียบเทียบ: สายคอนโทรล vs สายไฟทั่วไป

คุณสมบัติสายคอนโทรลสายไฟทั่วไป (Power)
วัตถุประสงค์ส่งสัญญาณ / คำสั่ง (24VDC, 4-20mA)ส่งพลังงานไฟฟ้าแรงดันสูง
โครงสร้างตัวนำฝอยละเอียด Class 5/6 ยืดหยุ่นสูงตัวนำแข็ง Class 1/2
จำนวน CoreMulticore 2-60+ Core มีตัวเลขกำกับ2-5 Core ระบุด้วยแถบสี
การป้องกันสัญญาณมักมี Shield (Braid/Foil)ไม่มี — เป็นแหล่งกำเนิดสัญญาณรบกวน

ประเภทสายคอนโทรลที่นิยมในอุตสาหกรรม

1. สาย Multicore (สายคอนโทรลทั่วไป)

ใช้เชื่อมต่อระหว่างตู้คอนโทรลไปยังเครื่องจักร — รุ่นยอดนิยม: YSLY-JZ, OPVC-JZ, JZ-500, Olflex Classic 110, CVV, VCT

2. สาย Shielded (สายคอนโทรลมีชีลด์)

จำเป็นเมื่อเดินสายใกล้ VFD/Inverter — รุ่นยอดนิยม: LiYCY, Olflex Classic 115 CY, CVV-S มี Braided Copper Shield ป้องกัน EMI

3. สาย Wiring (สายเดินภายในตู้)

สายเดี่ยวเดินวงจรภายในตู้ Switchboard — รุ่นยอดนิยม: H05V-K, H07V-K, VSF, THW-F

4. สาย Drag Chain (สายสำหรับรางกระดูกงู)

ออกแบบมาพิเศษสำหรับเครื่องจักรที่เคลื่อนที่ เช่น แขนหุ่นยนต์, CNC — ฉนวน PUR ทนการเสียดสี รองรับการโค้งงอนับล้านรอบ

วิธีเลือกสายคอนโทรลให้เหมาะกับการใช้งาน (4 ปัจจัย)

1. สภาพแวดล้อมสัญญาณรบกวน (EMI)

มี Inverter/VFD ใกล้เคียงหรือไม่? ถ้ามี "ต้อง" ใช้ สายชีลด์ (LiYCY, CVV-S)

2. ลักษณะการติดตั้ง: อยู่นิ่ง vs เคลื่อนที่

ติดตั้งถาวร → YSLY-JZ, CVV ได้ | เคลื่อนที่ → ต้องใช้สาย Drag Chain / Robot Cable

3. สภาพแวดล้อมทางกายภาพ

ในร่ม → PVC มาตรฐาน | เจอน้ำมัน/สารเคมี → PUR | กลางแจ้ง/UV → Black Sheath ทน UV

4. ขนาดพื้นที่หน้าตัดกับ Voltage Drop

ระบบ 24VDC ระยะไกล ควรขยับขนาดสาย (0.5 → 1.5 mm²) เพื่อลดความต้านทาน ป้องกัน Relay/Sensor ไม่ทำงาน

คำถามที่พบบ่อยเกี่ยวกับสายคอนโทรล (FAQ)

Q: สายคอนโทรลแบบมีชีลด์ดีอย่างไร?

มีชั้นตัวนำถักหรือฟอยล์ทำหน้าที่เป็นโล่ป้องกัน EMI จากมอเตอร์ขนาดใหญ่หรือ Inverter (VSD/VFD) ไม่ให้เข้ามาบิดเบือนสัญญาณ จำเป็นอย่างยิ่งในระบบ Analog (4-20mA)

Q: ใช้สายไฟทั่วไปแทนสายคอนโทรลได้ไหม?

ไม่แนะนำ เพราะสายคอนโทรลมีตัวนำฝอยละเอียด (Class 5/6) ยืดหยุ่นสูง มี "หมายเลข Core" สำหรับไล่วงจร เหมาะกับตู้คอนโทรลที่พื้นที่จำกัด ซึ่งสายไฟทั่วไปไม่มีคุณสมบัติเหล่านี้

Q: สายคอนโทรลทนความร้อนได้กี่องศา?

ฉนวน PVC มาตรฐาน (YSLY-JZ, CVV) ทน 70-80°C หากหน้างานร้อนกว่านี้ ควรใช้ Silicone (180°C) หรือ Teflon (260°C)

Q: เลือกขนาดสายคอนโทรลอย่างไร?

พิจารณา 2 ปัจจัย: (1) พิกัดกระแสต้องรองรับโหลดปลายทาง (2) Voltage Drop โดยเฉพาะ DC 24V ระยะไกล ควรขยับขนาดสาย (0.5 → 1.5 mm²)

Q: สายคอนโทรลใช้กับไฟบ้าน 220V ได้ไหม?

ได้ หากเป็นสาย Rated 300/500V ขึ้นไป แต่ต้องใช้ขนาดที่รองรับกระแสเพียงพอ สายคอนโทรลทั่วไปขนาด 0.5-1 mm² ไม่เหมาะกับโหลดกำลังสูง

Q: สายคอนโทรลใช้กับไฟ 3 Phase 380-400V ได้ไหม?

บางรุ่นเช่น YSLY-JZ 1kV หรือ CVV (600V) รองรับแรงดันได้ แต่ต้องคำนวณขนาดให้เหมาะกับกระแสจริง สายคอนโทรลไม่ได้ออกแบบมาเป็นสาย Power หลัก
